OpenWort is a very simple german thesaurus with many idioms and synonyms from <a href="http://www.openthesaurus.de/">OpenThesaurus</a> (LGPL licence).

Features:

✔️ works offline (maximal privacy)
✔️ no file access (maximal privacy)
✔️ very simple code for beginners
✔️ search and highlight results
✔️ click to copy an entry to clipboard
